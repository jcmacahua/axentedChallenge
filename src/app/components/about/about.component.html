Acerca de Blogs app

Se creo una app basica para administrar los bloggers favoritos. Teniendo las posibilidades de:

  • Agregar un nuevo blogger
  • Ver la lista de los bloggers
  • Asociar los bloggers con otros
  • Busqueda básica de bloggers

Para la presente app se utilizo las siguientes tecnologías

  • Angular CLI 15.1.3
  • Node 16.14.2
  • TypeScript 4.9.4
  • Bootstrap 5.3
  • Uso de LocalStorage
